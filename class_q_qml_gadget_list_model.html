Detailed Description

template<class ItemType>
class QQmlGadgetListModel< ItemType >

Provides a generic way to generate a list model from Q_GADGET derived class, suitable for QML.

QQmlGadgetListModel is a convenience subclass QAbstractListModel that makes use of C++ templates and Qt Meta Object to extract properties from a Q_GADGET derived class and create according roles inside the model.

This is a far better way than to expose directly a QList<Q_GADGET *> inside a QVariant.

And this is far simpler than doing all Qt model stuff manually : no subclassing or reimplementing need.

The class was designed so that most of the added API is really common with QList one.

Note : Simply needs that the class used for items inherits Q_GADGET and has Qt Meta Properties.
